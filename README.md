# SiShop - Flask Online Store

## Features
- User Authentication
- Product Management
- Order System
- SQLite Database
- Bootstrap 5 UI

## Installation

1. Clone project
2. Create virtual environment

python -m venv venv
source venv/bin/activate  (Mac/Linux)
venv\Scripts\activate     (Windows)

3. Install dependencies
pip install -r requirements.txt

4. Run
python app.py

5. Open browser
http://127.0.0.1:5000
